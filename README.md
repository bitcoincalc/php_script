# php_script
Php script to bitcoincalc

`check help`
```
Usage:
	btc <value>		check from btc convertions
	usd <value>		check from usd convertions
	vef <value>		check from vef convertions
	fee			check btc fee
	fee [bytes]		check btc fee for bytes
	wallet [public_key]	public key from wallet or multiple public keys separated by pipe character "|"
	price [source]		check prices for a diferent api (can be lbtc|cmc|dt)
	help		this message
```
